# GridCurveRenderer : [`CurveRenderer`](CurveRenderer.md)
### Namespace: Blackout.UI


Component that renders the curve on to a Graphic. This renderer differs from the button renderer as it scales the image to fit the entire curve whilst maintaining its position on the grid.

 ### Methods
| Type | Method | Description |
| --- | --- | --- |
| `void` | MarkDirty() | Mark the curve dirty so it will be recalculated |
